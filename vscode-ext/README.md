# CoreWar Redcode

CoreWar Redcode is a Visual Studio Code extension scaffold for editing and exploring Redcode warriors.

## Planned features

- Redcode language registration for `.red` and `.rc` files
- Syntax highlighting and editor configuration for Redcode source
- Completion, hover, and task providers for Redcode editing and CoreWar workflows
- Canvas-based battle viewer with play, pause, step, speed control, and multi-warrior loading
- Commands and tasks for opening a battle viewer, validating warriors, running battles, and building the WASM visualizer

## Development

This scaffold uses TypeScript and webpack for extension bundling.
